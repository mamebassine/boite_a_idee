Nom du projet
boite_a_idees collaborative

Description
Cette application de boîte à idées collaborative se veut à la fois élégante et conviviale,
 visant à simplifier l'organisation de la vie collaborative tout en encourageant la collaboration entre les utilisateurs.

Fonctionnalités
Possibilité pour l'utilisateur de créer des idées.

Capacité de modification des idées par l'utilisateur.

Option de suppression des idées proposées.

Consultation de toutes les idées proposées.

Affichage des idées en fonction des catégories.

Autorisation pour l'administrateur d'approuver ou de refuser une idée.

Configuration et Installation
Prérequis :

Vérifiez que votre machine dispose de PHP 8.2.10, 10.4.28-MariaDB et Apache/2.4.56 (ou un serveur web équivalent).
Configuration de la Base de Données :

Importez le fichier SQL boite_à_idées.sql dans votre base de données MySQL.
Configuration du Projet :

Clonez ce dépôt : git clone https://github.com/mamebassine/boite_a_idee.
Accédez au répertoire du projet : cd boite_a_idees.
Lancement de l'Application :

Démarrez votre serveur local.
Accès à l'Application :

Ouvrez votre navigateur et accédez à http://localhost/boite_a_idee.
Utilisation
Pour commencer, créez un compte sur l'application. Une fois connecté, vous accéderez à la page principale affichant toutes les idées proposées par les collaborateurs. Pour ajouter une idée, cliquez sur "Mes idées" dans le menu à droite. Vous pourrez ajouter, modifier ou supprimer vos idées sur la page correspondante.
